User Manual
1Open the App

Navigate to the WaveLite homepage (index.html). No login or signup is required.

2Enter Peer ID

Ask your friend to share their Peer ID or generate one for yourself. Input the Peer ID in the input field and click "Watch" to start listening for a connection.

3Share Your Own Peer ID

Click on the "Create" button to generate your own SDP and share it via any medium (email, chat). The other user will input your ID on their side.

4Connection Established

Once both sides have entered each other's IDs correctly, a peer-to-peer connection will be established automatically.

5Start Chatting

After connection, you can start messaging. Text, images, videos, audio files, documents, and even your location can be shared instantly.

6Use Gemini AI (Optional)

Click the AI button to switch to Gemini Assistant. You can ask it questions like a virtual chatbot. To use this, save your API key in Settings.

7Theme & Sound Settings

Customize your chat experience using the Settings modal. You can toggle between light and dark themes, enable or mute message sounds, etc.

8Exit or Reload Chat

If you want to go back or exit the current chat session, use the Refresh option in the ellipsis menu. This will reset the connection and return you to the start screen.
